# Hajri Admin Dashboard

Admin dashboard for managing departments, subjects, and timetables.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
